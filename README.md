# BMS College of Engineering Website

This is a responsive, interactive website built for **B.M.S College of Engineering**, showcasing:

- Home, About, Departments, Placements, Campus Life, and Alumni sections
- Interactive image slider on the homepage
- Placement logos and information
- Founders and history of the institution
- Chatbot to answer student queries about fees, departments, placements, skills, events, and opportunities
- Fully responsive design for desktop and mobile
- Built using HTML, CSS, and JavaScript

## Features
-Research, Campus Life, Alumni
- Student academics
- Chatbot for instant query resolution.
- Department cards
- About the Placement
- Responsive grid layouts for cards and sections
- Contact and application sections

## Future Improvements

- Upgrade chatbot with real AI using OpenAI API
- Student login and personalized dashboard
- Integration with backend database for real-time data
- More interactive campus tour and multimedia content

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Google Maps Embed for 360° campus tour
